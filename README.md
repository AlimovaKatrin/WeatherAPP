### How to start
#### 1️⃣ step
```
 git clone git@github.com:AlimovaKatrin/WeatherAPP.git

 cd WeatherApp
 
 yarn install
```
#### 2️⃣ step
Create `.env.local` file out of `src` repo. 
Insert following :
```
REACT_APP_API_KEY=df70b96788b60eb1c6312820a89a93f4
```
#### 3️⃣ step
```
yarn start
```
